# Netflix SQL Data Analysis Project

## Project Overview
This project analyzes Netflix content using SQL queries.

## Dataset
Netflix Titles Dataset (8807 records)

## Tools Used
- MySQL Workbench
- SQL

## Business Problems Solved

1. Count the number of Movies and TV Shows
2. Find the most common rating for Movies and TV Shows
3. List all movies released in a specific year
4. Find the top 5 countries with the most content
5. Identify the longest movie
6. Find content added in the last 5 years
7. Find all Movies/TV Shows by Rajiv Chilaka
8. List TV Shows with more than 5 seasons
9. Count content items in each genre
10. Find average release year by country
11. List documentary movies
12. Find content without a director
13. Find Salman Khan movies in last 10 years
14. Top actors in Indian movies
15. Categorize content as Good or Bad
